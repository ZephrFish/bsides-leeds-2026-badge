# Original firmware (upstream)

The unmodified badge firmware as shipped by punk-security, kept here for
reference so the custom build does not overwrite it. ZephrFish's custom version
lives in [`../zephrfish/`](../zephrfish).

| File | Description |
|------|-------------|
| `firmware.ino` | Original source (from upstream commit `64373d9`) |
| `firmware.hex` | Original Intel HEX image |
| `firmware.bin` | Original raw binary (8060 bytes) - identical to the repo's `../compiled/` |

These are provided as-is and are not built by CI.
